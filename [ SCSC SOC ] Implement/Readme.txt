## 창업 프랜즈
창업하는 데에 있어 필요한 정보를 제공해주는 어플리케이션이다. 
인구밀도, 소득 등 데이터를 이용하여 신규 입점 시 최적의 위치를 제공해주고
해당 위치와 브랜드에 관련된 시각적 정보를 제공한다.

## 설치 방법
안드로이드 스튜디오를 통해 해당 파일을 열고
컴퓨터와 핸드폰을 케이블로 연결한다. 컴퓨터의 소프트웨어정보에서
빌드 번호를 여러번 클릭하여 개발자 모드를 키고 USB 디버깅을 키고 다운 받는다.

## 사용 방법 
#구성
1. 회원 가입 - 사용자의 ID, PW, 이름, 전화번호, hint를 입력하여 사용자를 등록한다.

2. ID 찾기 - 기존 사용자의 이름, 전화번호, hint를 통해서 ID를 찾는다.

3. PW 찾기 - 기존 사용자의 ID, 이름, 전화번호, hint를 통해서 PW를 찾는다.

4. 로그인 - 가입한 ID와 PW를 입력하여 창업 프랜즈에 접속을 한다.

5. 메인화면 - MY, 카테고리, 검색/찾기, 즐겨찾기 로그아웃 등의 기능을 제공한다.

6. MY - 사용자의 정보를 조회, 수정, 탈퇴 기능을 제공한다..

7. 카테고리 - 업종별 프랜차이즈의 정보를 제공한다.
	     
8. map1 - 카테고리에서 선택한 프랜차이즈의 현재 입점 위치를 지도에 띄워준다. (대구 한정)

9. 검색/찾기 - 사용자가 프랜차이즈, 지역, 고려정도를 선택하여 추천 받은 위치와 즐겨찾기에 있는 같은 프랜차이즈를 지도에 띄워준다 (대구한정)

10. ShowData - 추천 받은 지역에 대한 인구밀도, 유입인구, 소득수준의 통계적 자료를 보여준다. (대구한정)

11. 즐겨찾기 - 추천 받은 위치 중 사용자가 추가한 위치에 대한 조회, 삭제, 갱신의 기능을 제공한다.

12. 로그아웃 - 로그아웃 버튼을 통해서 사용자가 창업 프랜즈 앱을 나갈 수 있다

# 상세
1. 회원가입 - 로그인 화면에서 회원가입 버튼을 누르고 사용자의 ID, PW, 이름, 전화번호, hint를 입력하고 save 버튼을 눌러서 가입을 한다.
	    ( Hint는 사용자만의 문자열, 숫자 등 아무거나 가능하다. )

2. ID 찾기 - 로그인 화면에서 ID 찾기 버튼을 누르고 사용자의 이름, 전화번호, hint를 입력하고 search 버튼을 누르면 해당 정보에 맞는 Id를 보여준다.

3. PW 찾기 - 로그인 화면에서 PW 찾기 버튼을 누르고 사용자의 ID, 이름 ,전화번호, hint를 입력하고 search 버튼을 누르면 해당 정보에 맞는 PW를 보여준다.

4. 로그인 기능 - 첫 화면에서 사용자가 ID와 PW를 입력하고 로그인 버튼을 누르면 DB와 비교해 main화면으로 전환된다.

5. 메인화면 - 로그인 후 들어가는 화면으로 MY, 카테고리, 검색/찾기, 즐겨찾기, 로그아웃 버튼이 있다.

6. MY - (조회) 메인화면에서 MY버튼을 눌러 Mypage화면으로 이동하면 해당 사용자의 이름, ID, PW, 전화번호, hint가 나온다.
           (수정) PW, 전화번호, hint text를 수정하고 수정버튼을 누르면 입력한 내용으로 사용자 정보가 수정이 된다.
           (탈퇴) 탈퇴 버튼을 누르면 해당 사용자의 정보가 삭제가 되면서 화면이 로그인 화면으로 전환된다.
            (홈)  우측 상단의 홈 버튼을 누를 경우 화면이 메인 화면으로 전환된다.

7. 카테고리 - (조회) 메인 화면에서 카테고리 버튼을 누르면 카테고리 화면으로 전환된다. 여기서 사용자가 보고 싶은 업종을 선택하면 해당 업종의 프랜차이즈의 목록이 나온다. (대구한정)
	     (홈) 우측 상단의 홈 버튼을 누를 경우 화면이 메인 화면으로 전환된다.

8. map1 - 카테고리 화면에서 선택한 프랜차이즈를 선택하면 map1화면으로 전환된다. 선택한 프랜차이즈의 현재 입점 위치와 정보를 같이 지도에 띄워준다. (대구한정)

9. 검색/찾기 - (search) 메인 화면에서 검색/찾기 버튼을 누를 경우 map2화면으로 전환된다. 사용자가 원하는 프랜차이즈, 지역, 고려정도를 선택하고 search버튼을 누르면 추천 위치와 함께 
		   즐겨찾기 중 이름이 같은 프랜차이즈 위치를 같이 띄워준다. 3가지(프랜차이즈, 지역, 고려정도)를 선택하지 않은 경우 검색이 불가능하다.
	      (stat) stat의 기능은 search를 하고 난 후 사용이 가능한 기능이다. 버튼을 누를 경우 showData화면으로 이동한다.

10. ShowData - 검색/찾기 화면에서 stat 버튼을 누를 경우 showdata화면으로 전환된다. 해당 지역의 인구밀도, 유동인구, 월평균 가구소득, 연령별 가구소득의 정보를 제공한다. (대구한정)

11. 즐겨찾기 - (조회) 메인 화면에서 즐겨찾기 버튼을 눌러 즐겨찾기 화면으로 전환되고 사용자가 추가한 즐겨찾기 목록이 나온다.
	      (삭제) 사용자가 추가한 즐겨찾기 중 삭제하고 싶은 목록의 하트 버튼을 누를 경우 삭제가 된다
	      (갱신) 우측 상단의 갱신 버튼을 누르면 즐겨찾기 목록이 갱신이 되면서 깔끔하게 정렬이 된다.

12. 로그아웃 - 로그아웃 버튼을 누를 경우 해당 사용자의 상태가 로그아웃 상태가 되며 로그인 화면으로 전환된다.
